## Setup .env example

Create .env file in your root project folder.

```env

DB_HOST = localhost
DB_USER = "your database username"
DB_PASSWORD = "your database password"
DB_DATABASE = payment
PORT = "your port"
SECRET_KEY = "your secret key"
BASE_URL = "your url"
EMAIL = "your email"
EMAIL_PASSWORD = "your password email"

```

## Installation
